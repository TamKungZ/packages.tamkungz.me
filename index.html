TamKungZ Maven

Public static Maven repository for TamKungZ projects.

Gradle

repositories {
    maven {
        name = "TamKungZ Maven"
        url = uri("https://maven.tamkungz.me/")
    }
}

Gradle Groovy DSL

repositories {
    maven {
        name = "TamKungZ Maven"
        url "https://maven.tamkungz.me/"
    }
}

Example dependency

dependencies {
    implementation "me.tamkungz:examplelib:1.0.0"
}

Maven artifacts are served from the root of this domain. Example path: /me/tamkungz/examplelib/1.0.0/examplelib-1.0.0.pom
